# Rollercoaster Ticket Program

A Python program that simulates a rollercoaster ticketing system.
It checks eligibility based on height, calculates ticket prices based on age, and adds optional photo charges.

# Features:

-- Height check (minimum 150cm required).
-- Age-based ticket pricing:
  - Child (<= 12 years): $5
  - Youth (<= 18 years): $7
  - Adult (>= 18 years): $10
-- Optional photo package (+$3).
-- Displays the final bill.
