# Number Guessing Game

A Python-based interactive game where the player tries to guess a randomly chosen number between 1 and 100.
The game includes two difficulty levels and tracks the number of attemps remaining.

# Features:

-- Randomly selects a number between and 100.
-- Two difficulty levels:
   - ** Easy ** -> 10 attemps
   - ** Hard ** -> 5 attemps
-- Provides hits ("Too high"/"Too low") after each guess.
-- Ends with either a win or loss message.
