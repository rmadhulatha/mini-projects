#Blackjack Game

A Python implementation of the classic Blackjack card game.
Play aganist the computer in the console, with rules for Blackjack, busts and score comparison.

#Features:
-- Deals random cards from a standard decks.
-- Handles Blackjack (Ace as 11 or 1).
-- Compares scores between player and computer.
-- Interactive gameplay with user input.
